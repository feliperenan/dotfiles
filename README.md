## Pre-requisites

* Homebrew for MacOS
* tools: iterm, nvim, tmux, rg, fzf
* Powerfonts: `https://www.nerdfonts.com/`

## Setup 

```bash
brew install nvim tmux rg fzf
git clone https://github.com/frenangomes/dotfiles
chmod +x setup.sh
./$HOME/dotfiles/setup.sh
```
